    // 03_03_blink_functrion_params

const int ledPin = 13;
const int delayPeriod = 250;

void setup() {
  pinMode(ledPin, OUTPUT);
}

void loop() {
  flash(20, delayPeriod);
  delay(3000);
}
// Podem veure que la funció te dos paràmetres que son 2 nombres enters i que s'anomenen "numFlashes" i "d"
// numFlashes no esta definit com a variable global, és una variable local definida dins de la funció "flash"
// i no te un valor predeterminat, només ha de complir que sigui un valor enter. El mateix per a la variable "d"
void flash(int numFlashes, int d) {
  for (int i = 0; i < numFlashes; i ++) {
    digitalWrite(ledPin, HIGH);
    delay(d);
    digitalWrite(ledPin, LOW);
    delay(d);
  }
}

Normas internacionals del codi Morse

  1. Una linia (dash) és igual a tres punts (dots)
  2. L'espai entre les parts d'una mateixa lletra és igual a un punt (dot) (S = ···, l'espai entre cada punt és un punt)
  3. L'espai entre dues lletres de la mateixa paraula són tres punts (dots) (Per exemple entre la S i la O de SOS)
  4. L'espai entre dues paraules és igual a set punts (dots) (Pa·······tomaquet)
